# 주요 기능
## 도서 관리
### 도서 조회
- 도서 조회는 누구나 가능
- 조회 조건은 도서 제목, 저자 명으로 가능하며, 차순정리는 발행년도, 도서제목으로 가능
- 간단 조회 결과 시 사용자에게 아래 내용 노출
  1. 제목
  2. 저자
  3. 발행년도
  4. 도서 상태
  5. 이미지
 
- 상세 조회 결과 시 사용자에 아래 내용 노출
  1. 제목
	2. 저자
	3. 발행연도
	4. 도서상태
	5. 이미지
	6. 리뷰 내용
	7. 내용 설명

## 도서 리뷰
 - 대여했던 도서에 한해 회원은 1회 후기 작성 가능
 - 도서 상세 조회 결과 시 해당 내용 노출
     
## 도서 대출/반납
 - 회원만 가능
 - 반납 기한은 대출일로부터 +7일
